ภาพรวมอัลกอริทึมเข้ารหัส

ยินดีต้อนรับสู่ Interactive Cryptography

แอปพลิเคชันนี้ถูกออกแบบมาเพื่อสรุปและสาธิตวิธีการเข้ารหัส (Encryption) และถอดรหัส (Decryption) ของ 7 อัลกอริทึมคลาสสิกที่สำคัญ เนื้อหาภายในแอปครอบคลุมตั้งแต่การแทนที่ตัวอักษรขั้นพื้นฐาน (Substitution) ไปจนถึงการสลับตำแหน่งที่ซับซ้อน (Transposition) คุณสามารถเลือกสำรวจแต่ละอัลกอริทึมผ่านเมนูด้านซ้าย เพื่อดูตัวอย่างทีละขั้นตอน กฎเกณฑ์ที่ใช้ และโต้ตอบกับข้อมูลเพื่อสร้างความเข้าใจที่ลึกซึ้งยิ่งขึ้น

ประเภทของ Cipher

  • 1
    Substitution (การแทนที่) แปลงตัวอักษรหนึ่งเป็นอีกตัวอักษรหนึ่ง (Caesar, Vigenère, Vernam, Playfair)
  • 2
    Transposition (การสลับตำแหน่ง) คงตัวอักษรเดิมไว้ แต่เปลี่ยนลำดับการเรียง (Route, Columnar, Double)
1

Caesar Cipher

Caesar Cipher คืออัลกอริทึมการเข้ารหัสแบบแทนที่ (Substitution) ที่เก่าแก่และเรียบง่ายที่สุด ทำงานโดยการเลื่อนตำแหน่งตัวอักษรในข้อความต้นฉบับ (Plaintext) ไปข้างหน้าหรือถอยหลังตามจำนวนค่าคงที่ที่กำหนดไว้เรียกว่า 'Key' ส่วนนี้จะสาธิตการทำงานผ่านการโต้ตอบด้วยตนเองและตัวอย่างการคำนวณแบบจำลองคณิตศาสตร์

Vigenère Cipher

ต่างจาก Caesar ที่ใช้ตัวเลขเดียว Vigenère Cipher ใช้ "คำ" เป็น Key ในการเข้ารหัส โดยนำตัวอักษรของ Key มาเทียบเป็นตัวเลข แล้วนำไปบวกกับข้อความต้นฉบับทีละตัว หาก Key สั้นกว่าข้อความ จะใช้วิธีนำ Key มาต่อกันแบบวนซ้ำ (Repeating Key) เพื่อให้ครอบคลุมความยาวทั้งหมด ส่วนนี้จำลองกระบวนการยืด Key และการคำนวณทีละตำแหน่ง

Vernam Cipher / XOR

Vernam Cipher ในระดับบิต (Binary) อาศัยตัวดำเนินการทางตรรกศาสตร์ที่เรียกว่า XOR (Exclusive OR) โดยนำบิตของ Plaintext มาทำ XOR กับบิตของ Key ข้อดีของ XOR คือการคำนวณมีความสมมาตร ใช้ฟังก์ชันเดียวกันทั้งการเข้ารหัสและการถอดรหัส (C ⊕ K = P)

กฎของ XOR Logic

0 ⊕ 0 = 0 เหมือนกันได้ 0
1 ⊕ 1 = 0 เหมือนกันได้ 0
1 ⊕ 0 = 1 ต่างกันได้ 1
0 ⊕ 1 = 1 ต่างกันได้ 1

คำนวณตามตัวอย่างรายงาน

P: 0110
K: 1011
C: 1101

พิสูจน์ถอดรหัส: นำ C(1101) มา XOR กับ K(1011) จะได้ P(0110) เหมือนเดิม

Columnar Transposition

ใช้วิธีนำข้อความมาเขียนเรียงเป็นแถวแนวนอนลงใต้คำที่เป็น Key จากนั้นลำดับการอ่านออกเพื่อเข้ารหัส จะถูกกำหนดโดยการเรียงลำดับพยัญชนะของ Key ตามหลักพจนานุกรม (A-Z) ซึ่งสร้างความซับซ้อนให้กับการจัดเรียงใหม่
